## 🌟 Features

- **Modern Design**: Clean, professional layout inspired by contemporary portfolio sites
- **Dark Theme**: Eye-catching dark UI with red accent colors
- **Responsive**: Fully responsive design that works on desktop, tablet, and mobile
- **Smooth Animations**: Fade-in effects, hover animations, and smooth scrolling
- **Interactive Elements**: Parallax blob effect, dynamic navigation highlighting
- **Fast Performance**: Lightweight and optimized for quick loading

## 📱 Sections

1. **Hero Section** - Eye-catching introduction with call-to-action buttons
2. **About** - Professional background and key stats (CGPA, internships, projects, awards)
3. **Experience** - Timeline of internships and professional roles
4. **Projects** - Featured projects with technologies and descriptions
5. **Skills** - Programming languages, cybersecurity tools, and platforms
6. **Certifications & Awards** - Professional achievements and recognitions
7. **Contact** - Easy-to-access contact methods (email, phone, LinkedIn, location)
